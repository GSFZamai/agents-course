# Introdução a Agentes

<img src="https://huggingface.co/datasets/agents-course/course-images/resolve/main/en/unit1/thumbnail.jpg" alt="Thumbnail"/>

Bem-vindo à primeira unidade, onde **você construirá uma base sólida nos fundamentos dos Agentes de IA**, incluindo:

- **Entendendo os Agentes**
  - O que é um Agente e como ele funciona?
  - Como os Agentes tomam decisões usando raciocínio e planejamento?

- **O Papel dos LLMs (Modelos de Linguagem de Grande Escala) em Agentes**
  - Como os LLMs atuam como o “cérebro” por trás de um Agente.
  - Como os LLMs estruturam conversas via o sistema de Mensagens.

- **Ferramentas e Ações**
  - Como os Agentes usam ferramentas externas para interagir com o ambiente.
  - Como construir e integrar ferramentas para o seu Agente.

- **O Fluxo de Trabalho do Agente:**
  - *Pensar* → *Agir* → *Observar*.

Depois de explorar esses tópicos, **você construirá o seu primeiro Agente** usando `smolagents`!

Seu Agente, chamado Alfred, realizará uma tarefa simples e demonstrará como aplicar esses conceitos na prática.

Você aprenderá até a **publicar o seu Agente no Hugging Face Spaces**, para que você possa compartilhá-lo com amigos e colegas.

Finalmente, no final desta Unidade, você fará um teste. Se passar, você **ganhará o seu primeiro certificado de curso**: o 🎓 Certificado de Fundamentos em Agentes.
